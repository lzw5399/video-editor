# Phase 10.1: Usable Editor MVP Completion - UI Spec

**Status:** Approved for implementation
**Language:** Simplified Chinese

## Visual Direction

Keep the Phase 04.1 Jianying-style professional workspace: compact top feature bar, left material/function panel, center preview monitor, right inspector, and bottom timeline. Do not add a landing page, tutorial card, decorative hero, or large explanatory copy.

## Required Interaction Contracts

- `导入素材` is a primary compact button in the media panel. It opens the system file chooser and does not require manual path entry.
- The media panel still shows search and kind/status filters. Imported items show thumbnail placeholder, file name, kind, duration, resolution or sample rate, and status badge.
- The preview monitor canvas remains black when no frame is available. When a frame is available, it shows the actual PNG inside the draft canvas aspect ratio.
- Artifact paths move to diagnostics/metadata below the canvas; the path must not be the primary preview content.
- Preview controls include current timecode, previous frame, next frame, stop, fit/ratio/fullscreen placeholders, and request-frame/request-segment buttons. Disabled controls remain visible.
- The timeline ruler and playhead line support pointer interactions without changing row heights or causing hover/selection layout jumps.
- Segment selection highlights the timeline block, shows a preview selection outline, and drives inspector content.
- Inspector no-selection state remains `草稿参数`. Selected video/image/text/audio state uses tabs `画面`, `音频`, `变速`, `动画`, `调节`, `AI效果`.
- Audio waveform P0 appears as a stable placeholder stripe in audio segment blocks; no real waveform cache is persisted.
- All user-visible copy uses Jianying-style Chinese terms: `草稿`, `素材`, `轨道`, `片段`, `播放头`, `预览`, `画面`, `音量`, `字幕`, `关键帧`.

## Layout And Density

- Preserve verified desktop viewports: `1280x800` and `1120x720`.
- Five regions must remain visible and non-overlapping: top feature bar, material panel, preview monitor, inspector, timeline.
- Scrollbars must stay compact/dark and not revert to default white scrollbars.
- Buttons and timeline controls must keep stable dimensions; hover and selection cannot resize toolbar rows, track rows, segment blocks, or preview canvas.
- Preview canvas uses CSS aspect ratio from `draft.canvasConfig.width / height`.

## Accessibility/Testability

- File import button accessible name: `导入素材`.
- Preview canvas accessible label: `预览画面`.
- Preview image accessible name: `当前预览帧`.
- Timeline ruler accessible label: `时间线标尺`.
- Playhead accessible affordance can be the visible line plus time inputs; tests may use CSS locators for drag geometry.
- Every icon-only button has Chinese `aria-label` and `title`.

## Boundaries

- Renderer may hold display-only local UI state such as active category, search/filter text, playhead position, and preview artifact URL.
- Renderer must not own canonical draft/timeline/render semantics.
- Any platform API added for dialogs must be narrow and separate from `videoEditorCore`.
- No copied Jianying assets, trade dress, icons, or proprietary resources.
