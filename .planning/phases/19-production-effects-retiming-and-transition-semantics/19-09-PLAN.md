## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Renderer UI to preload/main | Visible controls emit typed project intents and interactions. |
| UI ghost/proxy to native preview | Local overlays may show interaction feedback but cannot prove product pixels. |
| Desktop E2E evidence | Tests observe user-visible behavior and native preview/export facts. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-19-25 | Elevation of Privilege | renderer workspace | mitigate | Source guards and E2E reject renderer-owned draft mutation, time mapping, transition validation, effect evaluation, and FFmpeg strings. |
| T-19-26 | Repudiation | product preview evidence | mitigate | E2E requires native render-graph GPU evidence and no fallback success. |
| T-19-27 | Denial of Service | high-frequency UI controls | mitigate | Coalesced project interactions and revisionUnchanged assertions prevent save/undo loops per pointer move. |
| T-19-SC | Tampering | dependency stack | accept | No dependency changes. |
</threat_model>
