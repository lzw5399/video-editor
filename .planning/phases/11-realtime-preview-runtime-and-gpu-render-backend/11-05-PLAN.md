## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| preview service -> realtime runtime | Backend routing chooses realtime or fallback based on Rust-owned classification. |
| preview service -> FFmpeg artifact fallback | Unsupported paths may invoke existing FFmpeg artifact generation. |
| binding -> Electron main | Telemetry and diagnostics cross into desktop display as data, not fallback decisions. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-11-17 | Repudiation | fallback behavior | mitigate | Return serializable fallback reason, capability diagnostics, target time, and generation in every result. |
| T-11-18 | Denial of service | unsupported preview requests | mitigate | Classify before execution, reuse cache when available, and report fallback/cancellation telemetry. |
| T-11-SC | Tampering | package installs | mitigate | This plan adds no new external package beyond prior approved runtime dependencies. |
</threat_model>
