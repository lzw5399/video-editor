<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| draft snapshot -> engine/render graph | Accepted semantic draft is transformed into runtime render intent. |
| render graph -> realtime classifier | Renderer-neutral graph intent becomes backend capability decisions. |
| export graph -> preview parity diagnostics | Divergence must be explicit and serializable. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-04 | Tampering | graph preparation | mitigate | Use `engine_core` and `render_graph` APIs only; test no FFmpeg compiler path is called in graph preparation. |
| T-11-05 | Information disclosure | diagnostics | mitigate | Diagnostics include entity IDs, domains, support, and reasons only; do not include absolute local media paths beyond existing graph material URI behavior. |
| T-11-06 | Repudiation | parity divergence | mitigate | Snapshot serializable parity diagnostics for known divergence instead of relying on transient logs. |
| T-11-SC | Tampering | cargo dependencies | mitigate | This plan adds no unaudited package install; dependency changes remain limited to packages approved in `11-RESEARCH.md` if needed. |
</threat_model>
