<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| renderer to bindings_node | Untrusted UI requests cross into Rust command contracts. |
| preview_service to cache filesystem | Derived preview artifacts are written outside canonical draft JSON. |
| preview_service to media_runtime | Compiled FFmpeg jobs cross into process execution. |
| accepted timeline results to preview invalidation | Rust-owned edit results become cache invalidation inputs. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-09 | Tampering | preview cache | mitigate | Cache entries include target ranges, semantic fingerprints, output kind, and material dependencies; invalidation runs in `preview_service`, not renderer code. |
| T-05-10 | Elevation of Privilege | renderer preview UI | mitigate | Source guards block FFmpeg/render graph/cache key/script/process construction in renderer while allowing generated envelope builders. |
| T-05-11 | Information Disclosure | preview artifacts | mitigate | Store preview artifacts under service-owned cache roots and return bounded artifact metadata/URIs only; never persist cache paths in `.veproj/project.json`. |
| T-05-12 | Denial of Service | preview generation | mitigate | Classify runtime/capability/material errors and keep draft state unchanged; tests cover invalid requests and cache reuse. |
| T-05-SC | Tampering | npm/pip/cargo installs | mitigate | No new external packages; scripts guard package dependency drift and generated contract headers. |
</threat_model>
