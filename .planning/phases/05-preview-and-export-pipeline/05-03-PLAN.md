| Boundary | Description |
|----------|-------------|
| render_graph to ffmpeg_compiler | Renderer-neutral intents become FFmpeg job data. |
| compiler to media_runtime | Structured `Vec<OsString>` argument plans and sidecar definitions cross into runtime execution plans. |
| text layout profile to ASS sidecar | Resolved text overlays become derived subtitle artifacts. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-07 | Tampering | ASS/font generation | mitigate | Resolve deterministic font candidates, classify missing support, and snapshot ASS output including Simplified Chinese escaping. |
| T-05-08 | Information Disclosure | sidecar/filter scripts | mitigate | Treat scripts and ASS files as derived artifacts outside `.veproj/project.json`; source guards later block draft persistence. |
