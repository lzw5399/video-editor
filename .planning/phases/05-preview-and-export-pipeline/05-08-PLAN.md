<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| renderer to export commands | User-selected output path/preset crosses into Rust-owned validation and job planning. |
| binding registry to media_runtime | Electron-facing job IDs map to Rust runtime jobs. |
| export artifact to renderer | Validation and log summaries cross back as bounded display data. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-15 | Tampering | export command contracts | mitigate | Use Rust-generated serde/schemars/ts-rs contracts with command/payload pairing tests. |
| T-05-16 | Elevation of Privilege | export job registry | mitigate | Binding registry owns job IDs/status only; runtime owns process execution and cancel tokens. |
| T-05-17 | Information Disclosure | export logs/UI | mitigate | Display bounded log summaries and classified errors; never persist logs to `.veproj/project.json`. |
| T-05-18 | Repudiation | export visual checks | mitigate | Playwright saves export screenshots and asserts geometry at 1280x800 and 1120x720. |
| T-05-SC | Tampering | npm/pip/cargo installs | mitigate | No new external package installs; export UI uses existing React/Playwright stack. |
</threat_model>
