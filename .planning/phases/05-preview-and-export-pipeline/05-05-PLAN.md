| binding to preview_service | Binding path conversion and runtime composition call Rust preview generation/cache services. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-11 | Elevation of Privilege | binding routes | mitigate | Binding delegates preview semantics to `preview_service`; renderer receives artifact metadata only. |
| T-05-12 | Tampering | renderer helpers | mitigate | Source guards reject renderer-owned FFmpeg, render graph, cache key, invalidation, and process logic. |
| T-05-SC | Tampering | npm/pip/cargo installs | mitigate | No new external package installs; only local crates and already-approved workspace dependencies may be wired. |
</threat_model>
