<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Accepted draft -> audio_engine | Canonical Rust draft semantics enter DSP timeline evaluation. |
| Renderer/binding command -> audio session | Untrusted play, seek, pause, cancel, and status requests target Rust-owned audio sessions. |
| Audio session -> output trait | Bounded buffer metadata crosses from semantic runtime into output implementation. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-15-02-01 | Tampering | `AudioBufferRequest` | mitigate | Compare `PlaybackGeneration` on every buffer result; stale and canceled requests are non-presented and counted. |
| T-15-02-02 | Denial of Service | buffer/session contracts | mitigate | Bound buffer duration, channel count, frame count, cancellation token set size, and telemetry counters. |
| T-15-02-03 | Repudiation | audio diagnostics | mitigate | Return classified stale, canceled, missing-output, degraded-output, and unsupported-effect diagnostics with target time and generation. |
| T-15-02-04 | Elevation of Privilege / Information Disclosure | output trait boundary | mitigate | Public session/status results expose safe summaries only; native handles and raw buffers remain inside Rust output implementations. |
</threat_model>
