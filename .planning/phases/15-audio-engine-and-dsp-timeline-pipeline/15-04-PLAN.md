<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Electron renderer -> Node binding | Untrusted command payloads request audio sessions, device selection, and waveform data. |
| Binding registry -> audio_engine runtime | Opaque binding IDs map to private Rust session IDs and generation state. |
| Artifact store -> renderer waveform display | Derived waveform data crosses as bounded display-ready peaks only. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-15-04-01 | Spoofing / Tampering | opaque audio session IDs | mitigate | Prefix, length, ASCII-hex validation, unknown-session errors, and private runtime ID map. |
| T-15-04-02 | Information Disclosure | device and waveform responses | mitigate | Expose safe labels/status/peaks only; schema tests reject handles, stream configs, artifact roots, SQLite, blob paths, fingerprints, and dirty ranges. |
| T-15-04-03 | Denial of Service | waveform peak request | mitigate | Bound peak bin count and payload size in Rust before serialization. |
| T-15-04-04 | Repudiation | audio command results | mitigate | Return classified diagnostics for stale, canceled, malformed session, missing device, degraded output, and waveform unavailable states. |
</threat_model>
