## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-05-01 | Denial of Service | D3D frame leases | mitigate | Use frame-pool budgets and session-close cascading release diagnostics. |
| T-12-05-02 | Elevation of Privilege | D3D handles | mitigate | Return opaque IDs with owner session, generation, and device identity; no COM/D3D pointers cross bindings. |
| T-12-05-03 | Tampering | device compatibility | mitigate | Do not advertise texture-ready status unless Phase 11 device identity compatibility is proven; otherwise return CPU/fallback diagnostics. |
| T-12-05-SC | Tampering | Windows Cargo dependency | mitigate | 12-02B dependency verification must approve `windows` or approved equivalent before this plan relies on it. |
</threat_model>
