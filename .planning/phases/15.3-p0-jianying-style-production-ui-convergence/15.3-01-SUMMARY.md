## Decisions Made

- Normal launch is entry-only; explicit `VIDEO_EDITOR_TEST_WORKSPACE_FIXTURE=blank|demo` is the supported test/dev workspace bypass.
- The renderer does not create project semantics locally; it asks the platform for a bundle path and calls Rust-owned project commands.
- Product-safe project errors intentionally omit raw paths and stack/core diagnostics in the visible entry alert.
