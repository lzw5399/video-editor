| offline formula JSON -> `adapter_kaipai` | Untrusted sanitized formula evidence is deserialized into strict Rust contracts. |
| adapter contract -> generated schema/TypeScript | Generated artifacts can drift from Rust-owned contract semantics. |
| adapter input -> core/render crates | Raw provider concepts must not cross into canonical draft or render semantics. |

| T-03.1-01-PATH | Tampering | `FormulaResourceRef.path` | mitigate | Bundle contract represents resource evidence only; Plan 03.1-04 enforces path traversal rejection for localized renderable resources. |
