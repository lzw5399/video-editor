## Notes

- `RealtimePreviewCompositorOutput.render_backend` distinguishes `WgpuRenderPass` from `CpuReference`; CPU deterministic output remains test/reference-only evidence.
- Supported WGPU layers are sorted by render graph `stack_index`, then track/segment identity.
- External texture handles fail closed until explicit WGPU texture import exists.
- Bundled text support validates the repository font registry and glyph availability, then composites generated text textures in the same GPU render pass.
- This plan does not yet make the desktop preview window play imported media end-to-end. Real material-frame input and desktop presentation are owned by 15.2-03 and 15.2-04.
