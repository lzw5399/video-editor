## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Electron/native caller -> Rust project session | Caller requests a session-visible mutation. |
| adapter/import plan -> `.veproj/project.json` | Imported semantics become canonical project state. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-17-21 | Tampering | Project session revision | mitigate | Require `expectedRevision` and reject stale imports before applying or saving. |
| T-17-22 | Tampering | Atomic project write | mitigate | Apply validated plan then save through project_store in one success path; errors leave session state unchanged. |
| T-17-23 | Information Disclosure | Project JSON | mitigate | Tests inspect saved JSON for raw formula/provider IDs/remote runtime refs. |
| T-17-24 | Repudiation | Import response | mitigate | Return `AdaptationReport` with the same response as the updated view model. |
</threat_model>
