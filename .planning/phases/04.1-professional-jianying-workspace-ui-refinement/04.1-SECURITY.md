# Phase 04.1 — Security

> Per-phase security contract: threat register, accepted risks, and audit trail.

---

## Trust Boundaries

| Boundary | Description | Data Crossing |
|----------|-------------|---------------|
| Renderer component to App callbacks | Workspace controls collect user intent but must not execute Rust commands directly except through App-owned command paths. | UI intent, material/text/audio/timeline actions |
| Renderer display state to Rust-owned draft semantics | Search, filter, category, zoom, preview shell, inspector tabs, and visual metadata are display state only. | Draft/material/timeline state rendered read-only |
| Preview shell to Phase 05 render services | Preview/export UI slots are visible but must not construct render graph, FFmpeg, cache, or media artifacts. | Future preview/export intent only |
| Compact symbol controls to accessibility tree | Icon-like controls must retain Chinese accessible names and titles. | User-visible control labels |
| Package dependency boundary | Phase 04.1 uses dependency-free symbols and must not add icon packages or lockfile churn. | package manifests and renderer imports |
