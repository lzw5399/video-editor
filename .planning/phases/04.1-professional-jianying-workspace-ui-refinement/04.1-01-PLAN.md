<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| renderer component to App callbacks | Top/left UI collects user intent but must not execute Rust commands directly. |
| renderer display state to Rust-owned draft semantics | Search/filter/category state is local display state; draft/material mutations belong to Rust command responses. |
| icon-only controls to accessibility tree | Compact symbols must retain Chinese accessible names. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04.1-01 | Tampering | `FeaturePanel.tsx` material/text/audio controls | mitigate | Keep material/text/audio controls routed through existing props from `App.tsx`; source guard blocks direct `draft.tracks`, `track.segments`, timerange, undo/redo, and snapping mutation. |
| T-04.1-02 | Elevation of Privilege | renderer preview/export/media behavior | mitigate | Do not introduce FFmpeg, ffprobe, render graph, preview cache, thumbnail generation, waveform generation, proxy, or export script terms in renderer files. |
| T-04.1-03 | Information Disclosure | compact icon/symbol controls | mitigate | Every icon-like or tree/category button receives Chinese `aria-label` and `title`; visible copy remains Simplified Chinese. |
| T-04.1-SC | Tampering | npm installs | mitigate | No package install in this plan; dependency-free symbols avoid package legitimacy and lockfile churn. |
</threat_model>
