## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| timeline controls to App command handlers | Timeline can request edits through props only; App builds command envelopes. |
| display-only timeline derivations to Rust-owned semantics | View model may derive labels/classes but must not own edit, undo/redo, snapping, or timerange semantics. |
| timeline icon controls to accessibility tree | Compact toolbar/header symbols need Chinese accessible names. |
