<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| test harness -> generated media | Playwright helpers create deterministic media outside product code. |
| renderer -> command bridge | Product workflow must route edits, preview, and export through generated commands. |
| packaged app -> external FFmpeg | Packaged workflow uses env/PATH runtime and must classify failures. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-10 | Repudiation | no-mock workflow proof | mitigate | Command call log must include import, timeline edit, preview, export, and status commands with mocks disabled. |
| T-06-11 | Tampering | product runtime boundary | mitigate | Test helper FFmpeg usage is limited to fixture generation; source guards later reject product FFmpeg/process/render graph ownership. |
| T-06-12 | Denial of Service | packaged workflow | mitigate | Keep packaged workflow short and deterministic with bounded polling. |
</threat_model>
