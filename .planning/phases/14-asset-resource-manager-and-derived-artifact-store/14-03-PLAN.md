## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Source filesystem facts -> artifact validity | Source replacement/relink/delete facts can invalidate or retain derived blobs. |
| Phase 13 dirty contracts -> SQLite rows | Dirty ranges/domains/fingerprints cross into persistent artifact state. |
| Unknown dependency -> full-draft fallback | Ambiguous facts must fail closed without becoming the normal path. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
