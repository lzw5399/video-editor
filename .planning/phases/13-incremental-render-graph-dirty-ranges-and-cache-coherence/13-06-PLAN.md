## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| large fixture drafts -> graph/cache tests | Large deterministic drafts must not hide full invalidation regressions. |
| source guards -> renderer code | Guards enforce Rust-owned graph/cache/dirty/FFmpeg decisions. |
| runtime parity tests -> local FFmpeg availability | Runtime setup failures must be classified rather than silently skipped. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-13-16 | DoS | large-timeline gates | mitigate | Use bounded counts and structural assertions instead of machine-dependent timing. |
| T-13-17 | Tampering | renderer source | mitigate | Final source guard rejects renderer-owned dirty/cache/render graph/FFmpeg logic. |
| T-13-18 | Repudiation | final contract drift | mitigate | `pnpm run test:contracts` and `git diff --exit-code` prove generated contract consistency. |
| T-13-SC | Tampering | npm/pip/cargo installs | accept | No package installation is required. |
</threat_model>
