## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| command-specific edit modules -> shared delta builder | Local response helpers must not omit dirty facts. |
| undo/redo history -> cache coherence | Restored drafts may otherwise reuse stale derived artifacts. |
| material dependencies -> derived consumers | Material changes affect multiple tracks and artifact classes. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-13-07 | Tampering | command-specific delta emission | mitigate | Add coverage tests for every accepted mutating command and require non-empty semantic deltas. |
| T-13-08 | Tampering | undo/redo cache coherence | mitigate | Restore semantic state first, then emit deterministic ranges or full-draft invalidation. |
| T-13-09 | Information Disclosure | material dependency expansion | accept | Material IDs and ranges are already command contract data; no new secrets are exposed. |
| T-13-SC | Tampering | npm/pip/cargo installs | accept | No package installation is required. |
</threat_model>
