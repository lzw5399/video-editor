| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T20-02 | Tampering | `.veproj/project.json` | mitigate | Add tests that reject derived artifact/runtime/cache/export pollution and compare reopened canonical draft equality. |
| T20-03 | Information Disclosure | material URI paths | mitigate | Use repo-owned fixture paths for product tests and exclude absolute temp output paths from canonical summaries. |
| T20-SC | Tampering | npm/cargo installs | mitigate | No external package installs are planned; only move existing local `project_store` into `testkit` dependencies if the binary needs it. |
</threat_model>
