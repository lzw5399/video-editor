## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | generated schema/type files | mitigate | `just test` runs schema generation and `git diff --exit-code`. |
| T-01-02 | Repudiation | CI/local gate divergence | mitigate | CI runs `just build` and `just test`, not parallel bespoke commands. |
| T-01-03 | Spoofing/Tampering | FFmpeg availability in CI | mitigate | Install/probe FFmpeg and ffprobe for tests; render smoke remains required. |
| T-01-04 | Information disclosure | CI artifacts | mitigate | Do not upload release artifacts or package binaries in Phase 1 CI. |
| T-01-SC | Tampering | package installs | mitigate | Use lockfile-based `pnpm install --frozen-lockfile` and Cargo locked checks/tests. |
</threat_model>
