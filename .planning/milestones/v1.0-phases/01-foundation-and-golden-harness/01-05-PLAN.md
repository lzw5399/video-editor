## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| env/PATH -> media_runtime | User-controlled paths select FFmpeg/ffprobe candidates to probe. |
| media_runtime -> OS process | Rust launches local process candidates. |
| process stderr/stdout -> command envelope | External process output is summarized for logs/UI-ready errors. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Spoofing/Tampering | `VE_FFMPEG_PATH`, `VE_FFPROBE_PATH`, PATH lookup | mitigate | Probe explicit paths with `-version`, record source and checked paths, and surface structured errors. |
| T-01-02 | Denial of service | FFmpeg/ffprobe stderr/stdout | mitigate | Bound summaries with `MAX_STDERR_SUMMARY_BYTES` before returning errors. |
| T-01-03 | Tampering/Elevation of privilege | process execution | mitigate | Use `std::process::Command` argument arrays; grep for shell execution patterns. |
| T-01-04 | Repudiation | runtime probe command | mitigate | Return structured command errors with stable kinds and remediation text. |
| T-01-SC | Tampering | cargo dependency additions | mitigate | Use only 01-RESEARCH.md approved crates: `which`, `thiserror`, and existing workspace dependencies. |
</threat_model>
