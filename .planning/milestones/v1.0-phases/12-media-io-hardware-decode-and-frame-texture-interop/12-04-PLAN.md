| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-12-04-02 | Elevation of Privilege | Metal/CoreVideo handles | mitigate | Return opaque IDs with owner session, generation, and device identity; no native pointers cross bindings. |
| T-12-04-03 | Tampering | device compatibility | mitigate | Do not advertise texture-ready status unless Phase 11 device identity compatibility is proven; otherwise return CPU/fallback diagnostics. |

</threat_model>
