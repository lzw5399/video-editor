## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| media file -> FFmpeg process | Untrusted media bytes are decoded by an external process through bounded argument-array execution. |
| FFmpeg output -> frame pool | Process output and decoded bytes become a CPU frame lease and diagnostics. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-03-01 | Tampering | FFmpeg process arguments | mitigate | Use `FfmpegExecutor::run` with `Vec<OsString>` arguments and no shell-concatenated commands. |
| T-12-03-02 | Denial of Service | FFmpeg fallback decode | mitigate | Reuse bounded process timeout/output summaries and frame-pool budgets from 12-01. |
| T-12-03-03 | Repudiation | fallback selection | mitigate | Return selected path and classified reason for every fallback transition. |
| T-12-03-SC | Tampering | npm/pip/cargo installs | accept | This plan adds no package dependencies; it uses existing FFmpeg runtime dependencies. |
</threat_model>
