## Environment Notes

The existing Homebrew `ffmpeg` formula lacked `ass`, `subtitles`, and `drawtext` filters, which caused real text preview/export commands to fail in FFmpeg runtime execution. The machine now uses `ffmpeg-full 8.1.2`:

- `/opt/homebrew/opt/ffmpeg-full/bin/ffmpeg`
- `/opt/homebrew/opt/ffmpeg-full/bin/ffprobe`

`~/.zshrc` and `~/.zprofile` put `ffmpeg-full` first on PATH and set `VE_FFMPEG_PATH`/`VE_FFPROBE_PATH`. `scripts/phase15-3-desktop-gate.sh` also prefers `ffmpeg-full` when present.

Mac mini display note: product playback E2E requires an active macOS WindowServer with visible pixels. A real display or HDMI dummy, unlocked session, no screensaver/display sleep, and Screen Recording permission are required for screenshot-backed playback checks.
