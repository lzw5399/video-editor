## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] Tolerated FFmpeg `out_time_us=N/A` progress samples**
- **Found during:** Task 15.3-02-02 real workflow verification
- **Issue:** Real export failed because FFmpeg emitted `out_time_us=N/A`; `media_runtime` treated that sentinel as malformed progress and failed the export job.
- **Fix:** `media_runtime` now ignores `N/A` progress timestamp samples while preserving errors for genuinely malformed numeric values.
- **Files modified:** `crates/media_runtime/src/job.rs`, `crates/media_runtime/tests/export_job.rs`
- **Verification:** `cargo test -p media_runtime export_job_parses_progress_events`; full `tests/real-workflow.spec.ts`
- **Committed in:** `47a3801`

**2. [Rule 1 - Bug] Kept export path and preset refs current before command submission**
- **Found during:** Task 15.3-02-02 modal workflow verification
- **Issue:** Export command submission reads from `workspaceRef.current`; export path/preset UI changes updated React state but did not synchronously update the ref.
- **Fix:** `handleExportOutputPathChange` and `handleExportPresetChange` now update `workspaceRef.current` with the same next state returned to React.
- **Files modified:** `apps/desktop-electron/src/renderer/App.tsx`
- **Verification:** full `tests/real-workflow.spec.ts`
- **Committed in:** `47a3801`
