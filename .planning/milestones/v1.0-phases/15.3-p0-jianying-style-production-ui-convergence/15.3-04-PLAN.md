mistaken as complete unless the visible compositor frame actually changes.
</objective>

- T-15.3-04C: Product realtime errors expose backend/mock/cache internals. Mitigation: default status uses product-safe unavailable copy; detailed telemetry is developer-only.
</threat_model>
