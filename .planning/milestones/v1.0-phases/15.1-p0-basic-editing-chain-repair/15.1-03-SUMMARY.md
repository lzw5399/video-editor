# Phase 15.1 Plan 03: Bundled Font Registry Summary

**Baseline text preview and export now share a bundled, license-tracked CJK font registry entry with a stable `fontRef`.**

## Accomplishments

- Added bundled Noto Sans CJK SC Regular assets, OFL license material, README metadata, and Rust validation for file existence plus CJK glyph coverage.
- Changed default `TextFont` and desktop text insertion to `font://bundled/noto-sans-cjk-sc-regular`.
- Routed media runtime capability, Node bindings, FFmpeg ASS export, realtime preview text diagnostics, preview service fallback, and testkit parity capability through the bundled font registry.
- Refreshed generated command/draft schemas and desktop generated command result contract.

- `apps/desktop-electron/src/renderer/workspace/FeaturePanel.tsx` - Adds new text with bundled fontRef by default.

## Decisions Made

- Use Noto Sans CJK SC Regular as the baseline bundled font because it supports Simplified Chinese text and has clear OFL 1.1 license material.
- Keep broad font picker/product font management out of this plan; Phase 15.1 only needs deterministic baseline text parity.
- Preserve explicit unsupported handling for external/vendor `fontRef` values until an adapter/import compatibility layer maps them.
