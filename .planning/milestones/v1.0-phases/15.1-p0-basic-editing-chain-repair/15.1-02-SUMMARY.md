## Decisions Made

- Audio support is runtime capability data, not UI inference. The renderer only displays support/fallback facts returned by Rust-owned preview state.
- FFmpeg or cache fallback is no longer product copy for normal playback success. Default UI shows a playback-limited state; developer diagnostics keep the backend detail for debugging.
