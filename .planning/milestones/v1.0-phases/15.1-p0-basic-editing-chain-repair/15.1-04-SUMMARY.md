# Phase 15.1 Plan 04: First-Material Canvas Adaptation Summary

Empty drafts now adopt the first video/image material profile through Rust-owned timeline command semantics, and inserted visuals default to non-stretch `fit`.

## Accomplishments

- Added `CanvasAdaptationPolicy` to the draft canvas contract with backward-compatible serde defaults and generated TypeScript/schema updates.
- Added draft command canvas helpers so the first video/image material in an auto canvas draft sets width, height, aspect ratio, and video frame rate.
- Updated `addSegment` to emit a full-draft canvas invalidation delta plus `draftCanvasAutoAdapted` when auto adaptation occurs.
- Changed default `SegmentVisual` fit mode from `stretch` to `fit` across Rust model defaults and desktop test helpers.
- Added a committed portrait moving-video fixture and Playwright E2E covering blank draft import, add-to-timeline, accepted UI canvas readout, and selected overlay `data-fit-mode=fit`.
