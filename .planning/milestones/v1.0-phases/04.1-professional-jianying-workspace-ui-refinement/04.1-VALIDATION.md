## Sampling Rate

- **After every UI task commit:** Run `pnpm --filter @video-editor/desktop test:workspace` or the focused workspace spec if available.
- **After guard/package script changes:** Run `pnpm run test:phase4-source-guards` and any new Phase 04.1 guard script.
- **After every wave:** Run `pnpm run test:phase4-workspace`, `pnpm run test:phase4-source-guards`, and `pnpm run test:contracts`.
- **Before verification:** Run `just build && just test` when `just` is available. If local `just` is missing, run `pnpm run build && pnpm run test` and document the missing `just` blocker.
- **Max feedback latency:** Keep focused workspace/guard checks under 180 seconds.
