## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Playwright test harness to Electron app | Tests observe the real Electron bridge without replacing Rust command ownership. |
| renderer source to Rust-owned timeline semantics | Guards enforce that renderer does not own semantic mutation, undo/redo, snapping, or timeranges. |
| renderer source to Phase 05 media/render services | Guards enforce that Phase 04.1 UI shell does not implement preview/export/rendering. |
| compact icon controls to accessibility tree | Tests and guards enforce Chinese accessible names for icon-only controls. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-04.1-10 | Tampering | renderer timeline/material source | mitigate | Source guards block direct mutation of draft tracks, track segments, timeranges, undo/redo stacks, snapping, and MainTrackMagnet; Playwright command spy confirms edits flow through `executeCommand`. |
| T-04.1-11 | Elevation of Privilege | renderer preview/export source | mitigate | Source guards block FFmpeg, ffprobe, render graph/compiler, preview cache/frame, thumbnail, waveform, proxy, and export script terms in renderer files. |
| T-04.1-12 | Information Disclosure | icon-only controls | mitigate | Playwright verifies icon-only controls have Chinese `aria-label` and `title`; source guard prevents adding an icon dependency in this dependency-free implementation path. |
| T-04.1-SC | Tampering | npm installs | mitigate | No package installs; root guard fails if `lucide-react` appears in package files for this phase. |
