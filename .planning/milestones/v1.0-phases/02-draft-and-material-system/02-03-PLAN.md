## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| media path -> media_runtime | A local media path enters bounded ffprobe execution. |
| ffprobe JSON -> media_runtime | External process output is parsed into normalized metadata. |
| testkit generated media -> media_runtime tests | Temporary generated media exercises runtime parsing without committed binary fixtures. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-10 | Denial of Service | ffprobe process | mitigate | Use `FfmpegExecutor` argument arrays, existing timeout, bounded stdout/stderr summaries, and no shell strings. |
| T-02-11 | Tampering | ffprobe JSON parser | mitigate | Parse into normalized typed metadata only; reject malformed JSON with classified probe errors and never persist raw probe JSON. |
| T-02-12 | Tampering | Time/fps normalization | mitigate | Convert decimal durations into integer microseconds and fps strings into rational numerator/denominator values per D-07. |
| T-02-13 | Information Disclosure | Generated media fixtures | mitigate | Keep generated media in temp directories and assert no binary media is committed under fixtures/goldens. |
| T-02-SC | Tampering | npm/pip/cargo installs | accept | No new package-manager install is planned; existing serde_json is already approved in workspace. Stop for human legitimacy verification if a new dependency is proposed. |
</threat_model>
