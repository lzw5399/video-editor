- Product success evidence must not be satisfied by fallback, mock, artifact,
  CPU readback, debug, DOM, or legacy paths.
- Phase 18 introduces no desktop UI changes. Any future visible diagnostics
  require independent UI/design review before product acceptance.
- No new package upgrades are part of closeout. `cbindgen` remains pinned to
  `0.29.4`; `@napi-rs/cli` remains unchanged.
